# 高校宿舍管理系统 java961

## 项目概述

高校宿舍管理系统是基于Spring Boot+Vue的前后端分离项目，用于简化并自动化宿舍管理流程，提高工作效率。

## 技术栈

- **后端**：Spring Boot，Java
- **数据库**：MySQL
- **前端**：Vue.js，Element UI
- **项目管理工具**：Maven

## 功能模块

### 系统管理员

- **系统管理**：负责系统设置与维护
- **学生管理**：对学生信息进行管理
- **宿舍管理**：管理宿舍楼、宿舍分配等
- **考勤管理**：进行学生考勤记录的管理
- **出入登记管理**：管理学生的出入登记信息

### 宿舍管理员

- **学生缺勤登记**：记录学生的缺勤情况
- **学生缺勤记录查询**：查询并管理缺勤记录

## 系统角色

- **系统管理员**：负责系统的日常管理与维护
- **宿舍管理员**：负责宿舍区域内的日常管理

## 运行环境

- **后端**：Java 1.8+，Spring Boot 2.x
- **数据库**：MySQL 5.7+
- **前端**：Vue.js 3.x，现代浏览器
- **服务器**：推荐使用Linux服务器

## 部署步骤

1. 数据库环境搭建：部署MySQL，导入数据库脚本
2. 后端服务部署：编译后端代码，配置服务器环境，部署Spring Boot应用
3. 前端服务部署：编译前端代码，部署到Web服务器

## 核心亮点

- **前后端分离**：易于管理和维护，提高开发效率
- **响应式设计**：支持多种终端设备访问
- **模块化设计**：各个功能模块解耦合，便于扩展和维护
- **权限管理**：针对不同角色设定不同权限，安全性高
